TITLE: **The Student Marks Management System**
OVERVIEW:
 It is a Python-based console application designed to store and manage student academic records using a CSV file. The program features a user-friendly menu that allows users to perform complete CRUD operations such as
 1. Create
 2. Read
 3. Update
 4. Delete
FEATURES:
1. Menu-driven console interface
2. Written entirely in Python using built-in modules
3. Handles missing files and invalid inputs
4. Simple, modular code that is easy to understand and extend
5. Uses Python’s built-in csv and os modules
TECHNOLOGIES/TOOLS USED:
1. Python 3.13 – programming language used for implementation
2. CSV (Comma-Separated Values) – file format for storing student records
    csv Module – Used to read from and write data to the CSV file
    os Module – Used for file validation and deletion operations
3. Command Line / Terminal – User interface for menu-driven interaction
4. Text Editor or IDE – IDLE
STEPS TO INSTALL & RUN THE PROJECT:
Make sure Python 3.x is installed on your system
if not, download from: https://www.python.org/downloads/
Download Project as ZIP file and extract the content to new location
in GitHub, git clone https://github.com/adrutha25bce10630-dot/student-record-management-system.git
Navigate to the directory containing the .py file
Open Command Prompt / Terminal.
Navigate to the project location, then run the project
