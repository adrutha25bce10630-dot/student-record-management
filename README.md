The Student Marks Management System is a Python-based console application designed to store and manage student academic records using a CSV file. The program features a user-friendly menu that allows users to perform complete CRUD operations
**FEATURES:**
Stores student details and marks in a CSV file
Menu-driven console interface for easy use
Supports full CRUD operations:
Add new student records
View all stored records
Update marks for specific subjects
Delete individual or all entries
**Written entirely in Python using built-in modules**
Demonstrates practical experience with file handling and modular functions
Runs without any database or external dependencies
**Can be extended with additional features like GUI, search, sorting, or database support**
